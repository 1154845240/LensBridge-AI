# LensBridge AI 修改记录

## 2026-06-25

### 文件与隐私数据清理

- 修复删除会话后上传图片可能残留的问题。
- 删除会话、题目和全部历史统一使用安全文件清理逻辑，并返回清理结果。
- 服务启动时自动删除数据库未引用的孤儿上传图片。
- 移除旧截图目录测试、旧版上传流程测试和 Gemini 单模型临时诊断脚本。
- 清理本机构建缓存、Python 缓存、空目录、孤儿截图和中断遗留临时截图。

## 2026-06-24

### 模型回答格式统一

- 所有 Agent 共用同一套前端内容格式识别，不再按 Gemini、豆包等模型分别适配。
- SQL、JSON 和代码类纯文本自动转换为 Markdown 代码块，普通文字和已有 Markdown 保持原格式。
- 历史答案与实时流答案统一使用同一个内容规范化函数，避免同一回答在刷新前后样式不一致。
- SQL 代码块按主要子句自动换行；移动端代码块支持软换行，避免横向拖动才能阅读完整内容。

### 品牌与界面

- 项目由 GHG 更名为 LensBridge AI（镜桥 AI）。
- 完成桌面端和移动端响应式界面。
- 增加使用说明、实时截图状态和镜桥主题标识。
- 修复代码块溢出、顶部布局、底部安全区和 UTC 时间显示。
- 页面模型标签改为显示 Agent 别名。
- 全局配置移除多余英文括注。

### Windows 程序

- 将客户端与 FastAPI 服务整合为单进程。
- 构建无控制台、无任务栏、无托盘的单文件 EXE。
- 增加 Windows 单实例锁、滚动日志和优雅退出。
- 打包运行数据迁移至 `%LOCALAPPDATA%\LensBridgeAI`。
- 默认 F12 打开本机管理页面。
- EXE 可直接分享，无需安装 Python 或依赖。

### 截图与快捷键

- 支持单图和多图队列。
- 增加截图起点、15 秒倒计时、上传和错误状态。
- 快捷键支持组合键并实时生效。
- 增加快捷键冲突和包含关系校验。

### 模型与多端一致性

- 支持网页新增、编辑、删除和切换 Agent。
- 每条记录固定绑定实际生成模型。
- 重新分析使用当前模型并更新模型标签。
- 多个浏览器共享同一个后台生成任务，避免重复调用和答案不一致。
- SSE 改为订阅数据库中的统一生成结果。
- 增加模型流一致性测试。

### 数据与交互

- 支持会话创建、切换、重命名、删除和历史记录。
- 支持截图与文字提问。
- 支持 Markdown 渲染、代码高亮和图片预览。
- API Key 字段降低浏览器密码保存提示概率。
